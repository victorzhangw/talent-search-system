# 🚀 啟動說明

## 快速啟動（推薦）

### 方法 1：一鍵啟動

```
雙擊執行：START.bat
```

這會自動啟動前後端並開啟瀏覽器。

### 方法 2：分別啟動

如果一鍵啟動有問題，可以分別啟動：

1. **啟動後端**

   ```
   雙擊執行：start-backend.bat
   ```

   等待看到 "Uvicorn running on http://0.0.0.0:8000"

2. **啟動前端**

   ```
   雙擊執行：start-frontend.bat
   ```

   等待看到 "Local: http://localhost:3000"

3. **開啟瀏覽器**
   ```
   訪問：http://localhost:3000
   ```

## 測試腳本

### 測試後端設置

```
雙擊執行：test-backend.bat
```

這會檢查：

- Python 虛擬環境
- FastAPI 安裝
- API 文件
- 依賴完整性

## 首次使用

如果是第一次使用，需要先設置環境：

```
雙擊執行：setup-first-time.bat
```

這會：

1. 建立 Python 虛擬環境
2. 安裝後端依賴
3. 安裝前端依賴

## 可用的腳本

| 腳本                   | 用途                   |
| ---------------------- | ---------------------- |
| `START.bat`            | 一鍵啟動前後端（推薦） |
| `start-backend.bat`    | 只啟動後端             |
| `start-frontend.bat`   | 只啟動前端             |
| `test-backend.bat`     | 測試後端設置           |
| `setup-first-time.bat` | 首次環境設置           |
| `venv-start-all.bat`   | 完整版啟動（含檢查）   |

## 服務端口

- **後端 API**: http://localhost:8000
- **前端界面**: http://localhost:3000
- **API 文檔**: http://localhost:8000/docs

## 常見問題

### Q: 啟動後畫面卡住？

A: 使用 `START.bat` 或分別啟動 `start-backend.bat` 和 `start-frontend.bat`

### Q: 後端啟動失敗？

A: 執行 `test-backend.bat` 檢查設置，如有問題執行 `setup-first-time.bat`

### Q: 前端啟動失敗？

A: 確認 `frontend\node_modules` 存在，如不存在執行：

```
cd frontend
npm install
```

### Q: 端口被佔用？

A: 關閉佔用端口的程序，或修改配置文件中的端口號

## 停止服務

關閉對應的 CMD 視窗即可停止服務。
