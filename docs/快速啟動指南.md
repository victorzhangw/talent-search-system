# 快速啟動指南

## 🚀 一鍵啟動（推薦）

### 方法 1：使用啟動腳本

雙擊執行：

```
start_all.bat
```

這會自動：

1. ✅ 啟動後端 API 服務（http://localhost:8000）
2. ✅ 在瀏覽器中開啟前端介面
3. ✅ 顯示系統狀態

### 方法 2：手動啟動

#### 步驟 1：啟動後端

開啟第一個命令提示字元視窗：

```cmd
cd BackEnd
start_api.bat
```

等待看到：

```
Application startup complete.
Uvicorn running on http://0.0.0.0:8000
```

#### 步驟 2：開啟前端

直接用瀏覽器開啟：

```
talent-chat-frontend.html
```

## 🛑 停止系統

雙擊執行：

```
stop_all.bat
```

或手動關閉後端命令提示字元視窗。

## 📊 系統狀態檢查

### 檢查後端是否運行

訪問：http://localhost:8000

應該看到：

```json
{
  "message": "人才聊天搜索 API",
  "version": "1.0.0"
}
```

### 檢查 API 文檔

訪問：http://localhost:8000/docs

應該看到 Swagger UI 介面。

### 檢查前端連接

開啟前端後，右上角應該顯示：

```
🟢 已連接
```

## 🎯 開始使用

在聊天框中輸入：

```
我需要一個善於溝通的銷售人員
```

系統會：

1. 🤖 LLM 分析需求
2. 🔍 從資料庫查詢特質
3. 📊 生成 SQL 查詢
4. 👥 返回匹配的候選人
5. 💡 顯示推薦理由

## 📁 檔案結構

```
專案根目錄/
├── start_all.bat              # 一鍵啟動腳本
├── stop_all.bat               # 停止系統腳本
├── talent-chat-frontend.html  # 前端介面
│
└── BackEnd/
    ├── talent_search_api.py   # 後端 API
    ├── start_api.bat          # 啟動後端
    ├── requirements_api.txt   # Python 套件
    └── venv/                  # 虛擬環境
```

## ⚙️ 配置

### 修改後端端口

編輯 `BackEnd/talent_search_api.py` 最後一行：

```python
uvicorn.run(app, host="0.0.0.0", port=8000)  # 修改 port
```

### 修改前端 API 位址

編輯 `talent-chat-frontend.html`：

```javascript
apiBaseUrl: 'http://localhost:8000'  # 修改為你的後端位址
```

## 🔧 疑難排解

### 問題 1：後端無法啟動

**症狀**：執行 start_all.bat 後沒有反應

**解決**：

1. 確認虛擬環境已建立：

   ```cmd
   cd BackEnd
   python -m venv venv
   ```

2. 安裝套件：

   ```cmd
   venv\Scripts\activate
   pip install -r requirements_api.txt
   ```

3. 再次執行 start_all.bat

### 問題 2：前端顯示「未連接」

**症狀**：前端右上角顯示紅色「未連接」

**解決**：

1. 確認後端正在運行：訪問 http://localhost:8000
2. 檢查瀏覽器控制台是否有 CORS 錯誤
3. 確認防火牆沒有阻擋 8000 端口

### 問題 3：端口被占用

**症狀**：錯誤訊息 "Address already in use"

**解決**：

1. 找出占用端口的進程：

   ```cmd
   netstat -ano | findstr :8000
   ```

2. 終止該進程：

   ```cmd
   taskkill /PID <進程ID> /F
   ```

3. 或修改後端端口（見上方配置說明）

### 問題 4：資料庫連接失敗

**症狀**：後端日誌顯示 SSH 連接錯誤

**解決**：

1. 確認 SSH 私鑰檔案存在：

   ```
   BackEnd/private-key-openssh.pem
   ```

2. 檢查資料庫配置：
   編輯 `BackEnd/talent_search_api.py` 中的 `DB_CONFIG`

3. 測試連接：
   ```cmd
   cd BackEnd
   venv\Scripts\activate
   python test_llm.py
   ```
