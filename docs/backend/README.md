## 🔧 測試工具

### 數據庫測試

```bash
# 測試數據庫連接和結構
cd tests
python test_fixed_search.py

# 測試 JSONB 查詢
python test_jsonb_queries.py
```

### 搜索功能測試

```bash
# 測試搜索引擎
cd BackEnd
python talent_search_api.py

# 在另一個終端測試
curl -X POST http://localhost:8000/api/search \
  -H "Content-Type: application/json" \
  -d '{"query": "找一個溝通能力強的人"}'
```
