人才搜索系統 - 啟動腳本說明
================================================================
創建日期: 2025-11-15

已創建的啟動腳本：
================================================================

1. 人才搜索系統.bat ⭐ (推薦)
   - 主控制面板
   - 包含所有功能的選單
   - 可以啟動、停止、檢查狀態
   - 可以打開測試頁面和文檔

2. 啟動人才搜索服務.bat
   - 直接啟動 API 服務
   - 在當前視窗運行
   - 可以看到即時日誌
   - 按 Ctrl+C 停止

3. 啟動並測試.bat
   - 在後台啟動服務
   - 自動打開測試頁面
   - 適合快速測試

4. 停止服務.bat
   - 停止所有運行中的服務
   - 清理 Python 進程

================================================================
使用方法：
================================================================

最簡單的方式：
1. 雙擊 "人才搜索系統.bat"
2. 選擇 [2] 啟動服務並打開測試頁面
3. 等待瀏覽器自動打開
4. 開始測試！

================================================================
服務地址：
================================================================

API 服務: http://localhost:8000
API 文檔: http://localhost:8000/docs
健康檢查: http://localhost:8000/health

================================================================
測試方式：
================================================================

1. 使用測試頁面: test_api.html
2. 使用聊天界面: talent-chat-frontend.html
3. 使用 API 文檔: http://localhost:8000/docs
4. 使用 curl 命令測試
